Mensagem secreta é um sistema para se comunicar de forma confidencial e segura, utilizado para e-mails e mensagens instantâneas. Utiliza a famosa Cifra de César que utiliza um sistema de deslocamento. Para utilizar é simples basta escrever sua palavra/frase e escolher o número de deslocamento, sendo da esquerda a direita, Ex.: se você colocar a letra 'A' com deslocamento '3' o resultado será 'D'. E para decifrar basta fazer o mesmo cole a cifra que recebeu e informe qual foi o deslocamento utilizado.
